These are the source files containing C code for compiling the program spei calculator. spei.c contain the main code, and the rest of files contain additional functions which are used as includes by the main code.

For more information on what these programs do, examples files and binaries for Windows and Mac OS X, please refer to http://hdl.handle.net/10261/10006. Additional information on the SPEI index can be found at http://sac.csic.es/spei/spei_index.html and the papers referred therein.
